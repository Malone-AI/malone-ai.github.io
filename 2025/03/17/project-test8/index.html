1.2 提供后端接口

先需要提供后端的批量导出接口，由前端调用。

Controller层：

通过输出流的方式导出。

  1. 调用后台服务查询所有管理员数据（adminService.selectAll()）。
  2. 使用 Hutool 的 ExcelUtil 创建一个 ExcelWriter 对象。
  3. 通过 writer.addHeaderAlias 设置表头别名，将数据库字段名映射为中文表头。
  4. 调用 writer.write(admins) 将查询得到的数据写入 Excel 文件。
  5. 设置响应头，指定文件名和 MIME 类型，使浏览器识别为文件下载。
  6. 获取响应输出流，并将 Excel 文件数据写入输出流，最后刷新并关闭流。

这种设计使得前端通过访问 /admin/export 接口即可触发文件下载。

  • exportData 为什么添加 throws Exception：

exportData 方法涉及到文件输出、URL编码、流操作等，很可能抛出 IO 或编码相关异常。如果不在方法签名上声明 throws Exception，就必须在方法内部捕获并处理这些异常。添加 throws Exception 可让异常由全局异常处理器统一管理，同时简化方法内部的异常处理逻辑。
